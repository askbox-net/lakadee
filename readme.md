# Lakadee Service
[Lakadee](https://www.lakadee.com.la/)

## Use

* Flask (Obvious!)
* Flask-SQLAlchemy (ORM for database)
* Flask-WTF (Generation of forms and validations)
* Faker (Generates fake data)

## Install

```bash
pip3 install -r requirements.txt
python3 migrations.py
```
## Run

```bash
python3 app.py
```
